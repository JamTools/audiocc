# audiocc

Audio Collection Cleanup

## Usage

```
Usage: audiocc [OPTIONS] PATH

Positional Args:
  PATH           directory path

Options:
  -artist string
      treat as specific artist
  -collection
      treat as collection of artists
  -force
      probes all files, even if path looks good
  -version
      print program version, then exit
  -write
      write changes to disk
```
